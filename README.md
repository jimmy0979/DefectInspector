## 資料庫輸入

1. 先以 ./GPM_Data/data.py 運行生成測試資料
2. 安裝 SQL Server 2019 與其 GUI 介面 SSMS(SQL Server Management Studio)
3. 新增一個 Database 並命名其為 test
4. 於 test 按右鍵 "Tasks" 再右鍵 "Import Flat File"
5. 將資料匯入

## 專案運行

1. 以 Visual Studio 2019 開啟本專案 (需先安裝 CLR)
2. 進入專案屬性更改 OpenCV 檔案連結位置 (詳細請看 OpenCV_Setup.docx)
3. 按下 connect 連線，連線約等待 40~45 秒
4. 點選右上方 Mapper 格子點，即可隨時更新主畫面 Painter 的晶粒資料
